## Projeto de Criatividade Computacional
(Você deve usar uma API key do GPT para poder rodar a aplicação)

Essa aplicação tem a finalidade de transforma imagens de produtos em verdadeiras imagens de conversão instantanea de tráfego.
Ao fazer o upload de uma imagem de um produto, selecionar gatilhos psicologicos de venda na geração da imagem, e opcionalmente adicionar uma descrição para ela. 
A aplicação deve gerar uma imagem  que chamativa e visualmente atraente para quem busca vender produtos usando meios digitais, como redes sociais. 
Tudo isso feito com embasamento psicologico sobre cores e imagens que prendem a atenção de seus usuarios ou que gere uma atração maior que imagens comuns, tudo isso sem a ajuda de um profissional.
